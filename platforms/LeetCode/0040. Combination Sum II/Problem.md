# 40. Combination Sum II

> Medium

Given an integer array `candidates` and a target integer `target`, return *a list of all unique combinations* of `candidates` where the chosen numbers sum to `target`.  
Each number in `candidates` **may only be used once** in the combination.  
The solution set **must not contain duplicate combinations**. You may return the combinations in any order.

### Example:
- Input: `candidates = [10,1,2,7,6,1,5], target = 8`  
- Output: `[[1,1,6],[1,2,5],[1,7],[2,6]]`

- Input: `candidates = [2,5,2,1,2], target = 5`  
- Output: `[[1,2,2],[5]]`

### Constraints:
- `1 <= candidates.length <= 100`
- `1 <= candidates[i] <= 50`
- `1 <= target <= 30`
