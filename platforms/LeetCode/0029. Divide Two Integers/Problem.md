# 29. Divide Two Integers

> Medium

Given two integers `dividend` and `divisor`, divide the two integers **without using** multiplication (`*`), division (`/`), or modulo (`%`) operators.  
Return *the quotient* after dividing `dividend` by `divisor`. The integer division should **truncate toward zero**, meaning the fractional part is discarded.
### Example:
- Input: `dividend = 10, divisor = 3`  
- Output: `3`  
- Explanation: `10/3 = 3.33333…`, truncated to `3`.
- Input: `dividend = 7, divisor = -3`  
- Output: `-2`  
- Explanation: `7/-3 = -2.33333…`, truncated to `-2`. 

### Constraints:
- `-2³¹ <= dividend, divisor <= 2³¹ - 1`
- `divisor != 0`
