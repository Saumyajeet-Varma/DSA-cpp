# 19. Remove Nth Node From End of List

> Medium

Given the `head` of a linked list, remove the **nth node from the end** of the list and return *its head*.

### Example:

![removenthnodefromendoflist](https://assets.leetcode.com/uploads/2020/10/03/remove_ex1.jpg)

- Input: `head = [1,2,3,4,5], n = 2`  
- Output: `[1,2,3,5]`

### Constraints:
- The number of nodes in the list is in the range `1 <= sz <= 30`
- `0 <= Node.val <= 100`
- `1 <= n <= sz`
