# 55. Jump Game

> Medium

You are given a **0-indexed** integer array `nums` where `nums[i]` represents the **maximum jump length** from that position.  
Return `true` if you can **reach the last index**, or `false` otherwise.

### Example:
- Input: `nums = [2,3,1,1,4]`  
- Output: `true`

- Input: `nums = [3,2,1,0,4]`  
- Output: `false`

### Constraints:
- `1 <= nums.length <= 10⁴`
- `0 <= nums[i] <= 10⁵`
