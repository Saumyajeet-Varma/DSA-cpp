# 69. Sqrt(x)

> Easy

Given a non-negative integer `x`, compute and return *the integer part* of the square root of `x`.  
In other words, return the **floor** of `sqrt(x)` — the greatest integer `r` such that `r * r <= x`.

### Example:
- Input: `x = 4`  
- Output: `2`

- Input: `x = 8`  
- Output: `2`  
- Explanation: The square root of `8` is `2.82842…`, and the integer part is `2`.

- Input: `x = 0`  
- Output: `0`

### Constraints:
- `0 <= x <= 2³¹ - 1`
