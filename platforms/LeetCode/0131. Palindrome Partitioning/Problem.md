# 131. Palindrome Partitioning

> Medium

Given a string `s`, return *all possible palindrome partitioning* of `s`.  
A **palindrome partitioning** is a decomposition of the string such that **every substring** in the partition is a palindrome.  
You may return the answer in **any order**.

### Example:
- Input: `s = "aab"`  
- Output: `[["a","a","b"],["aa","b"]]`

- Input: `s = "a"`  
- Output: `[["a"]]`

### Constraints:
- `1 <= s.length <= 16`
- `s` consists of **lowercase English letters**
