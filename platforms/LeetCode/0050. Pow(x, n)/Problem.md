# 50. Pow(x, n)

> Medium

Implement `pow(x, n)`, which calculates `x` raised to the power `n` (**i.e.,** `xⁿ`).

### Example:
- Input: `x = 2.00000, n = 10`  
- Output: `1024.00000`

- Input: `x = 2.10000, n = 3`  
- Output: `9.26100`

- Input: `x = 2.00000, n = -2`  
- Output: `0.25000`  
- Explanation: `2⁻² = 1 / (2²) = 0.25`

### Constraints:
- `-100.0 < x < 100.0`
- `-2³¹ <= n <= 2³¹ - 1`
- `n` is an integer
- Either `x` is not zero or `n > 0`
- `-10⁴ <= xⁿ <= 10⁴`
